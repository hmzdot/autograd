# autograd.zig

Implementation of [automatic differentiation](https://pytorch.org/tutorials/beginner/blitz/autograd_tutorial.html)
in Zig.

## Features

### Tensor
- [x] `Tensor` struct
- [x] `Tensor` creation
- [x] `Tensor` addition
- [x] `Tensor` multiplication
- [ ] Broadcasting

### Autograd
- [x] Computation graph
- [x] Backpropagation for addition
- [x] Backpropagation for multiplication
